3. Unity Import Guide (Asset sprites are alredy configured)
Step 1 — Import

Drag the PNG into your Unity project.

Step 2 — Texture Settings

Select the sprite and set:

Texture Type: Sprite (2D and UI)

Sprite Mode: Multiple

Pixels Per Unit: 16

Filter Mode: Point (No Filter)

Compression: None

Generate Mip Maps: Off

Click Apply.

Step 3 — Slice the Sprite Sheet

Open Sprite Editor

Click Slice

Set:

Type: Grid by Cell Size

Cell Size: 16 × 16

Offset: 0

Padding: 0

Click Slice → Apply

Each item will now be a separate sprite.

Pixel Art Best Practices

Do not scale sprites using non-integer values

Use integer scaling only (×2, ×3, ×4, etc.)

Keep Point filtering enabled

Avoid color space conversion or compression. This ensures colors remain exactly as authored.
